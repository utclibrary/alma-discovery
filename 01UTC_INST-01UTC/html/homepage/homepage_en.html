 How does this work?

UTC Library Search provides simple, one-stop searching for books and e-books, videos, articles, digital media, and more.

UTC Library Search also helps you manage your research. Sign-in in order to:

  • Renew books and videos
  • Create favoriets lists
  • Export citations to Endnote Web
  • View full search results. (Some databases, only show results when you're signed in.)
